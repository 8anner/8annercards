## 1. 游戏概览

《大乱斗》是一款以三国杀为基础、融合多种游戏灵感而诞生的双人对战卡牌游戏。玩家各自选择一名角色，通过打出卡牌来造成伤害、防御攻击、使用特殊能力，目标是将对手的生命值降至0。

| 项目 | 内容 |
| --- | --- |
| 游戏人数 | 2人 |
| 游戏类型 | 回合制 PVP |
| 胜利条件 | 使对手生命值归零，并完成死亡判定 |

## 2. 游戏准备

### 2.1 选择角色

每位玩家从角色牌中 **选择一名角色**，或自行协商决定。角色决定后将角色牌置于自己面前，记录初始生命值。

### 2.2 整理牌堆

将所有非角色卡牌洗匀，作为公共抽牌堆。若玩家需要记录卡组构成，可自行整理牌表。

### 2.3 初始手牌

游戏开始时，每位玩家从牌堆 **摸4张牌** 作为起始手牌。

### 2.4 血量记录

用纸笔或其他方式记录每位玩家的当前生命值。

## 3. 卡牌类型

一张卡牌可以同时属于多种类型，以下为四大基础分类：

| 类型 | 定义 |
| --- | --- |
| **攻击牌** | 可以造成伤害的牌 |
| **防御牌** | 可以规避伤害的牌 |
| **功能牌** | 具有特殊能力的牌（非攻击、非防御） |
| **装备牌** | 描述中写有“可装备”的牌，打出后置于装备区，持续生效 |

**注：** 一张牌可以同时跨越多种类型（例如【孤独的风】既是攻击牌也是功能牌）。

## 4. 回合流程

每回合按照以下顺序进行：

### 阶段一：法力恢复阶段

你的法力值 **恢复到上限3点**。（关于法力值系统，详见第7节）

### 阶段二：摸牌阶段

从牌堆 **摸1张牌**。（此数量可能受角色技能或卡牌效果影响）

### 阶段三：行动阶段

你可以进行以下操作，顺序不限：

-   **打出手牌**：支付卡牌左上角标注的法力值（费），打出卡牌并执行效果。
-   **装备卡牌**：将装备牌打出并置于装备区。
-   **使用角色技能**：根据角色描述发动技能。

**关于攻击牌的限制：**  
每回合可造成的伤害不再设有上限。但遵循以下规则：

> 你可以打出一张可造成伤害的牌，伤害无上限。  
> 如果此次造成的伤害 **小于1.5**，你可以再打出一张攻击牌，但总伤害不能超过1.5。  
> 每回合最多打出 **2张** 攻击牌。

*(源自2026-02-01更新)*

### 阶段四：结束阶段

-   结算所有“直到回合结束”的效果。
-   若手牌数量超过 **手牌上限**，需弃牌直到符合上限。
-   **基础手牌上限：4张**（可能受角色技能及装备影响）。

## 5. 伤害与防御

### 5.1 伤害类型

| 伤害类型 | 说明 |
| --- | --- |
| **近战伤害** | 普通攻击造成的伤害，可以被防御牌格挡 |
| **魔法伤害** | 无视普通防御的特殊攻击，直接造成生命值减少。在描述中用 **蓝色** 标注 |
| **任意伤害** | 近战伤害与魔法伤害的统称，简写为“伤害” |

### 5.2 防御结算

-   一张防御牌 **只能防住一次伤害**。
-   **魔法伤害不受普通防御影响**，除非有特殊说明（如装备【水晶球】后可用【防御】规避魔法伤害）。
-   对于多次伤害（如【致命球】），需要多次防御。

**举例：**  
【致命球】造成1.0近战伤害 + 0.5魔法伤害。对手必须分别用：

-   一次防御抵挡1.0近战伤害
-   另一次防御（或特殊手段）应对0.5魔法伤害

### 5.3 攻击防御抵消机制

部分攻击牌可被视为防御牌使用：

> 任意一张攻击牌或防御牌，均可用来抵挡攻击牌造成的伤害。  
> 使用攻击牌抵消伤害时，只取其“防御端”效果，不造成伤害。

## 6. 装备规则

### 6.1 装备槽

每位玩家默认拥有以下装备槽：

| 槽位 | 可装备数量 |
| --- | --- |
| 武器 | 1 |
| 防具 | 1 |
| 饰品 | 2 |

【过载】装备后，可以增加一个任意种类的装备槽位。

### 6.2 装备特性

-   装备牌打出后，置于装备区，效果持续生效。
-   部分装备为 **一次性**，效果触发一次后立即弃置（如【稻草人】）。
-   **一次性装备不受到角色技能加成**（如骑士的“失去装备时摸一张牌”）。

## 7. 法力值系统

*（于2026-04-07更新引入）*

### 7.1 核心概念

-   每位玩家拥有法力值，**上限为3点**。
-   打出手牌时，需支付卡牌左上角标注的 **法力值费用（费）**。
-   法力值在每个自己的回合开始时 **自动补满至3点**。

### 7.2 关于费用

-   **0费牌**：可以自由打出，不消耗法力值。
-   **多费牌**：需有足够法力值才能打出。
-   法力值用完后，无法再打出需付费的牌（除非通过其他手段恢复法力值，如【放血】、【不屈】）。

## 8. 关键术语

| 术语 | 释义 |
| --- | --- |
| **规避** | 等同于“格挡”或“防止”，使伤害不造成血量减少 |
| **弃置/丢弃** | 将卡牌从手牌或装备区移入弃牌堆 |
| **移出游戏** | 卡牌永久离开本局游戏，不再被任何玩家抽取或使用 |
| **一次性** | 装备触发一次效果后立即弃置 |
| **主动造成伤害** | 玩家通过打出手牌直接造成的伤害（区别于角色技能或装备被动触发的伤害） |
| **生命垂危** | 生命值 ≤ 1.0 的状态 |
| **不受到加成** | 卡牌效果不受角色技能影响（特指“一次性”装备） |
| **无上限** | 一张牌造成的伤害不受数值封顶限制 |

## 9. 死亡判定

当一名玩家生命值归零时，该玩家 **不会立即死亡**：

> 玩家必须 **操作完当前完整的回合**，回合结束后若生命值仍为0，才判定为死亡。
